---
title: "Get Connection"
description: "Retrieve details of a specific WhatsApp connection."
---

`GET /api/connections/{id}`

Returns the full details of a single WhatsApp connection by its ID. The connection must belong to the authenticated user.

## Authentication

| Header | Type | Required | Description |
|--------|------|----------|-------------|
| `Authorization` | string | Yes | Bearer token. Format: `Bearer <token>` |

## Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `id` | string | Yes | The unique identifier of the connection. |

## Response

```json title="200 OK"
{
  "id": "c1a2b3c4-d5e6-7f89-0abc-def123456789",
  "userId": "your-user-id",
  "workerId": "w9a8b7c6-d5e4-3f21-0fed-cba987654321",
  "sessionName": "u_a80ffc98_s_c1a2b3c4",
  "phoneNumber": "+1234567890",
  "status": "online",
  "engine": "NOWEB",
  "createdAt": "2026-03-14T10:00:00.000Z",
  "updatedAt": "2026-03-14T10:05:00.000Z"
}
```

```json title="401 Unauthorized"
{
  "statusCode": 401,
  "message": "Unauthorized"
}
```

```json title="404 Not Found"
{
  "statusCode": 404,
  "message": "Connection not found"
}
```
