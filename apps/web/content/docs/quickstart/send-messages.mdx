Send text messages to any WhatsApp contact or group through a connected number.

## Send a message

<Tabs items={["TypeScript", "Python", "CLI", "cURL"]}>
  <Tab value="TypeScript">
```typescript
const result = await client.sendMessage(
  connectionId,
  '1234567890@s.whatsapp.net',
  'Hello from Wago!'
);
console.log(result.id); // message ID
```
  </Tab>
  <Tab value="Python">
```python
result = client.send_message(
    connection_id,
    chat_id="1234567890@s.whatsapp.net",
    text="Hello from Wago!",
)
print(result["id"])
```
  </Tab>
  <Tab value="CLI">
```bash
wago connections send CONNECTION_ID 1234567890@s.whatsapp.net "Hello from Wago!"
```
  </Tab>
  <Tab value="cURL">
```bash
curl -X POST https://api.wago.com/api/connections/CONNECTION_ID/send \
  -H "Authorization: Bearer wh_..." \
  -H "Content-Type: application/json" \
  -d '{"chatId": "1234567890@s.whatsapp.net", "text": "Hello from Wago!"}'
```
  </Tab>
</Tabs>

## Chat ID format

| Type | Format | Example |
|------|--------|---------|
| Individual | `{phone}@s.whatsapp.net` | `1234567890@s.whatsapp.net` |
| Group | `{id}@g.us` | `120363419246216126@g.us` |

<Callout type="info">
  Use the [chats endpoint](/docs/api-reference/connections/chats) to discover chat IDs
  for existing conversations.
</Callout>

## Anti-spam protection

Wago automatically follows WhatsApp anti-blocking best practices when sending messages:

1. **Mark as seen** — acknowledges the conversation
2. **Start typing** — shows typing indicator
3. **Random delay** — 1-3s base + ~50ms per character (max 8s)
4. **Stop typing** — clears indicator
5. **Send message** — delivers the text

This happens transparently — you just call `sendMessage` and the delay is handled server-side.

<Callout type="warn">
  WhatsApp may block numbers that send too many messages too quickly or send
  unsolicited messages. Always respect WhatsApp's terms of service.
</Callout>
